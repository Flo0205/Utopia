Welcome to Utopia.

What is Utopia

Utopia is a API thats based on Atlas (© 2019 Luke Janik).
Utopia implements a couple of events and methodes to simply access the potential of Atlas.
Beyond that it fixes some small performance issues to reduce lag.

Is Utopia open source?

No.
Utopia isn't open source and is also obfuscated.
If you have security concerns you can ask if some Northwood Studios has checked the files.
If you are from Northwood Studios, I can send you the source code under the condition that it didn't gets moddified or published.
